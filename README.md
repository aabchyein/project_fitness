#### ■ 주요 기능
**<사용자>**
- 회원가입 -> 검색 (지역&업체) -> 예약 -> 마이페이지 -> 예약 조회 및 수정, 삭제
- 마이페이지(회원 기본정보 확인 및 수정)
- 후기 작성 및 별점 적용(업체 위치를 네이버맵에서 확인)

**<관리자>**
- 회원관리, 예약관리, 게시판관리, 공지사항 수정/삭제, 관리자 설정
